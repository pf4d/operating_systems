Description:
  A program which demonstrates the proper use
  of POXIX threads and semaphores.
  
Usage:
  
  1. Compile with in L/Unix with $make
  
  2. run with ./prodcon <run time> <num producers> <num consumers>
     where:
       <run time> is the time for the process to run in seconds,
       <num producers> is the number of producers to create, and
       <num consumers> is the number of consumers to create.
  
  3. Watch the thing work.
