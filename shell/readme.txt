Description:
  Shell program takes input and gives output.
  
  
Usage:
  
  1. Compile with in L/Unix with $make
  
  2. Enter in L/Unix commands with '!' prefix
  
  3. Other commands are:
  
     user        - print user name
     uid         - print user id
     quit or q   - quit
